1.安装node.js

Hexo是基于nodeJS编写的，所以需要安装下nodeJS和里面的工具npm

安装方法：直接前往官网下载LTS版本的即可。

因为npm是node官方的包管理器，因为网络原因有时可能无法使用，这里最好安装下cnpm（即国内版npm）

2.安装Git

Git是目前世界上最先进的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。廖雪峰老师的Git教程。Git教程

下载安装：直接到Git的官网进行下载即可。
安装完成后在cmd窗口中使用 git –version 命令来验证下是否安装成功。

Git安装完成后要进行一些基本的信息设置：

1、设置用户名：git config – global user.name ‘你再github上注册的用户名’;
2、设置用户邮箱：git config – global user.email ‘注册时候的邮箱’;

注意：该配置会在github主页上显示谁提交了该文件

3、配置完成后使用以下命令检查是否配置成功

注意：git config –global 参数，有了这个参数表示你这台机器上所有的git仓库都会使用这个配置，当然你也可以对某个仓库指定不同的用户名和邮箱

5.配置部署时免密登录

前面我们执行 hexo d 这个部署命令时，要输入我们的账号和密码，以后每次进行部署时显然很麻烦，接下来我们就进行免密的配置。

首先生成 ssh key密钥，打开git bash执行下面这行命令

一路回车即可，这是会在你的电脑C盘Users/账户名/这个文件夹下有一个.ssh文件，里面就是生成的密钥，打开带有.pub后缀的这个文件，复制里面的密钥。

然后登陆你的github帐户。点击你的头像，然后 Settings -> 左栏点击 SSH and GPG keys -> 点击 New SSH key

输入完密钥后，在git bash测试密钥是否正常工作：

$ ssh -T git@github.com

如果看到

Hi xxx! You’ve successfully authenticated, but GitHub does not # provide shell access.

说明引用成功。
